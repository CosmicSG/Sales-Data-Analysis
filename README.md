# 📊 Sales Data Analysis

## 📌 Project Overview

This project analyzes retail sales data using Python to uncover business insights, identify sales trends, and evaluate product performance.

The analysis includes data cleaning, visualization, exploratory data analysis (EDA), and business recommendations.

---

## 🎯 Objectives

- Clean and preprocess sales data
- Analyze sales performance
- Identify top-selling products
- Visualize trends using charts
- Generate business insights

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

- sales_data.csv
